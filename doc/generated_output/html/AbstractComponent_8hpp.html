Detailed Description

Abstract for all components.

The abstract component is used to describe each component within the system. This abstract includes all of the custom attribute functionality, which allows for more dynamic types to be included to describe a component.

Components can also store children of itself, which means you can develop a hierarchy of the same component. How this hierarchy is built depends on the type of component.

To summarize, components are containers that can fit into a tree hierarchy. These containers hold dynamic types through the CustomAttribute type.
